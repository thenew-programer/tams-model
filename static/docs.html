The TAMS API provides machine learning-powered anomaly prediction for industrial equipment. Get started by testing the health endpoint:

üéØ Key Endpoints

POST /predict/single - Predict scores for a single anomaly

Submit equipment number, system, and description to get AI predictions.

POST /predict/batch - Process multiple anomalies

Send an array of anomalies for bulk processing.

POST /predict/file/csv - Upload CSV file

Upload a CSV file with multiple anomaly records.

POST /predict/file/excel - Upload Excel file

Upload an Excel file (.xlsx, .xls) with anomaly data.

GET /anomalies - List anomalies

Retrieve paginated list of processed anomalies.

GET /anomalies/{id} - Get specific anomaly

Retrieve details for a specific anomaly by ID.
